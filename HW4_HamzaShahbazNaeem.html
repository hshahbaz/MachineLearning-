#QUESTION 2

#A) Assume that there are n observations in the sample. Since Bootstrap replaces the observations there are n-1 observations that are not the jth observation. This means we have n-1 observations in a total of "n" which gives us the fraction (n-1)/n.

#B) (n-1)/n. The concept remains the same as part A since we draw with replacement.

#C) The probability that the jth observation is not from the sample is (n-1)/n. Since we draw with replacements, we get the probability ((n-1)/n)^n.

#D) replacing n by 5 in ((n-1)/n)^n we get ((5-1)/5)^5  = 0.32768, which is the probablity of not having the jth observation. We subtract the number from 1 to get (1-0.32768) =0.67232 as the probability that the jth observation is in the bootstrap sample.

#E) 1−((100-1)/100)^100 = 0.634

#F) 1−((1000-1)/1000)^1000 = 0.632

def loo(a, b):
    ypred = result.predict(x)
    xbar = np.mean(x, axis=0)
    xsum = np.sum(np.power(x - xbar, 2))
    nrows = np.shape(x)[0]
    err = []
    for row in range(0, nrows):
        hi = (1 / nrows) + (np.sum(x[row] - xbar) ** 2 / xsum)
        mse = (y[row] - ypred[row]) ** 2 / (1 - hi)
        err.append(mse)
    return np.mean(err)
